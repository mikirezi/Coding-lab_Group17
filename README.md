## Project Objective

This project simulates a secure hospital monitoring infrastructure for Kenyatta National Hospital (KNH). The system handles patient vitals, water reserve monitoring, security permissions, log analysis, and log archiving using Linux shell scripting.

## Features

- Secure permissions using chmod
- Critical patient monitoring
- ICU water reserve analysis
- Automated log rotation
- Git collaboration using branches
